Обробка масивів та колекцій JavaScript

Мета роботи

Розглянути приклади використання коду JavaScript в HTML-документі. Навчитись опрацьовувати масиви та колекції. Навчитись отримувати доступ до об'єктів DOM.

Обладнання

Персональний комп'ютер. Текстовий редактор Sublime Text 3 або Notepad++. Пакет програм XAMPP. Web-браузер Chrome, Firefox, Opera, Internet Explorer*

Теоретичні відомості

Хід роботи

  1. Перевірити чи встановлено пакет програм web-розробника XAMPP
  2. Викликати панель керування xampp-control.exe
  3. Впевнитись, що web-сервер Apache запущений
  4. Перейти за адресою http://127.0.0.1/ або http://localhost/ та впевнитись, що сторінка вітання XAMPP завантажилась
  5. Очистити зміст каталогу C:\xampp\htdocs\
  6. Створити файл index.html в середині каталогу C:\xampp\htdocs\
  7. Файл index.html зробити з порожнім вмістом секції <body>
  8. Створити файл script.js в середині каталогу C:\xampp\htdocs\. Під'єднати script.js в index.html за допомогою тегу <link>
  9. У "підвал" сторінки помістити інформацію про виконавця роботи: група, ПІБ, дата виконання.
  10. Для кожного етапу роботи зробити знімки екрану або скопіювати текст консолі та додати їх у звіт з описом кожного скіншота
  11. Зберегти звіт у форматі PDF
  12. Роздрукувати звіт та письмово відповісти на контрольні запитання

  4. Яку дію виконує метод document.write()?
  5. Яку дію виконує метод console.log()?
